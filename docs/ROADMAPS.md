# AI Engineering Roadmaps

## RAG

1. Document ingestion and chunking strategy
2. Embedding selection and indexing
3. Retrieval tuning and reranking
4. Prompt grounding and citations
5. Evaluation and regression checks

## Agents

1. Tool contracts and schema design
2. Planning loop and guardrails
3. Memory and state handling
4. Human-in-the-loop checkpoints
5. Observability and rollback strategy

## Fine-tuning

1. Dataset design and curation
2. Data cleaning and de-duplication
3. Train/validation/test split
4. Offline and online evaluation
5. Safety and policy review

## MCP

1. Define capabilities and tools
2. Implement MCP server contracts
3. Add auth, limits, and sandboxing
4. Client orchestration patterns
5. Monitoring and incident handling

## Vector Databases

1. Pick index type and metadata strategy
2. Chunk size and overlap experiments
3. Filtered retrieval and hybrid search
4. Recall/precision benchmarking
5. Capacity planning and sharding

## Evaluation

1. Build golden datasets
2. Add automatic scoring and LLM judges
3. Track latency, token usage, and cost
4. Run A/B tests for prompts and models
5. Gate releases on eval thresholds

## MLOps

1. Version prompts, models, and datasets
2. CI checks for templates and pipelines
3. Deployment workflows with canary releases
4. Telemetry dashboards and alerting
5. Automated rollback and recovery plans
